Categories - список категорий продуктов (соусы, мясо, напитки … )
Contacts - список контактов (поставщики, потребители, работники)
	   (с должностью, адресом … )
CustomerCustomerDemo - пустая (идентификатор типа и идентификатор клиента)
CustomerDemographics - пустая (предполагаются клиенты (номер и описание))
Customers - список клиентов (компания, контактное лицо … )
Employees - список сотрудников (персональные данные)
EmployeeTerritories - список локаций закрепленных за сотрудниками по ID
Order Details - детали каждого заказа (продукт, цена, скидка)
Orders - заказы (отправка, поступление, перевозчик …)
Products - список продуктов (наименование, ед. измерения, упаковка)
Region - список регионов (север, юг, …)
Shippers - список грузоотправителей
Suppliers - поставщики (контактн. лицо, адрес …)
Territories - список городов (id, привязка к региону)
